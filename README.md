Website untuk mengelola data matakuliah dengan fitur CRUD, Sistem ini dilengkapi dengan login admin, dashboard, dan fitur pencarian data. Cara Penggunaannya :   
(1) Import file database.sql ke database MySQL bernama akademik,   
(2) Copy semua file ke folder htdocs  
(3) Akses via browser http://localhost/crud-matakuliah/login.php.   
(4) Login menggunakan username: admin dan password: admin. Jika terjadi masalah session (langsung masuk dashboard), akses file clear_session.php terlebih dahulu untuk membersihkan session.
